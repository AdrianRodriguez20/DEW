¿Qué es JavaScript?

JavaScript es el lenguaje de programación encargado de dotar de mayor interactividad y dinamismo a las páginas web.
¿Para qué sirve JavaScript?

Con este lenguaje de programación del lado del cliente (no en el servidor)

podemos crear efectos y animaciones sin ninguna interacción, o respondiendo

a eventos causados por el propio usuario tales como botones pulsados
